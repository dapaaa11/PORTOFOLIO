# Optimization Strategy

## Frontend Optimization

Use:

* Dynamic imports
* Route-based splitting
* Lazy loading
* Optimized rendering

---

## Media Optimization

Use:

* Adaptive bitrate streaming
* AVIF/WebP images
* Deferred video loading
* CDN caching

---

## Animation Optimization

Prefer:

* GPU transforms
* Opacity animation
* Lightweight motion

Avoid:

* Heavy repaint effects
* CPU-intensive animation
